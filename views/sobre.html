Sobre

Descrição

Este projeto é uma aplicação web para gerenciamento de eventos de tiro esportivo. Ele utiliza Node.js para o backend e HTML/CSS para o frontend.

Estrutura do Projeto

O projeto segue os princípios da Clean Architecture, dividindo o código em diferentes camadas:

  • Presentation: Contém os controladores e as views.
  • Domain: Contém a lógica de negócios.
  • Infrastructure: Contém a configuração do servidor e as rotas.

Instalação

Para executar o projeto em live-server, execute no terminal:

nodemon index.js

Para instalar novos módulos, execute:

npm install <nome-do-pacote>

Uso

  1. Clone o repositório: 
    git clone <URL-do-repositório>
  2. Navegue até o diretório do projeto: 
    cd Tiro_Esportivo
  3. Instale as dependências: 
    npm install
  4. Inicie o servidor: 
    nodemon index.js
  5. Abra o navegador e acesse: 
    http://localhost:3000

Dependências

  • Node.js
  • Express
  • Nodemon

Regras de Negócio

Autenticação de Usuários

  • Usuários podem se cadastrar utilizando um email e senha.
  • Usuários podem se autenticar utilizando suas credenciais (email e senha).
  • Integração com Google OAuth para permitir login com contas Google.
  • Sessões de usuário devem expirar após um período de inatividade.

Relatórios

  • Relatórios de eventos podem ser gerados em formato PDF.
  • Relatórios devem incluir informações detalhadas sobre os eventos, participantes e resultados.
  • Relatórios de uso do sistema, incluindo número de logins, eventos criados e outras métricas relevantes.

Funcionalidades do Sistema

  • Cadastro e gerenciamento de eventos de tiro esportivo.
  • Cadastro e gerenciamento de participantes.
  • Sistema de pontuação e ranking para eventos.
  • Notificações por email para eventos e atualizações importantes.
  • Dashboard administrativo para visualização de métricas e relatórios.

Valores de Mensalidade para Clubes

  • Mensalidade básica: R$ 100,00 por mês.
  • Pacote trimestral: R$ 270,00 (10% de desconto).
  • Pacote semestral: R$ 510,00 (15% de desconto).
  • Pacote anual: R$ 960,00 (20% de desconto).
